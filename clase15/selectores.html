Uso Descripción
X[att^=”val”] Elemento X cuyo atributo comience con "val"
X [att$=”val”] Elemento X cuyo atributo termine con "val"
X [att*=”val”] Elemento X cuyo atributo contenga el término "val"
X:checked Elemento X que esté checked
X:disabled Elemento X que esté deshabilitado
X:enabled Elemento X que esté habilitado
X:empty Elemento X que esté vacío o no tenga hijos
X:first-of-type Elemento X que sea el primer hermano de su tipo
X:last-child El último hijo de un elemento X
X:last-of-type Elemento X que sea el último hermano de su tipo
X:nth-child(n) Elemento X que sea el hijo número (n) de su padre
X:nth-last-child(n) Elemento X que sea el hijo número (n) de su padre, contando desde el último
X:nth-of-type(n) Elemento X que sea el número (n) de hermano
X:nth-last-of-type(n) Elemento X que sea el número (n) de hermano, contando desde el último
X:not(s) Elemento X que no coincida con el selector s
X:only-child Elemento X que sea el único hijo de su padre
X:only-of-type Elemento X que sea el único hijo de su padre de su tipo
X::selection Parte de X que sea seleccionada por el usuario
X:target Elemento X que sea el target de una dirección URL
X ~ Y Elemento Y que esté precedido por un elemento X
